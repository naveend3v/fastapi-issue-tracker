# Issue Tracker
Learning fastapi with this "Issue tracker' demo project
## Steps to reproduce
- Clone the repo to your laptop
    - ```git clone https://github.com/naveend3v/fastapi-issue-tracker.git```   
- Open terminal / cmd and navigate to the working directory & create python virutal environment
    - ```python -m venv .venv```
- Activate the virual environment
    - If operation system is linux execute below command.
        - ```source ./.venv/bin/activate```
    - If operation system is windows execute below command.
        - ```.venv\Scripts\activate```
    - Ref: https://www.w3schools.com/python/python_virtualenv.asp
- Install python dependencies
    - ```pip install -r requirements.txt```
- Run the fastapi application:
    - ```fastapi dev main.py```
